### **Ways to Use Callbacks in JavaScript & TypeScript**
A **callback function** is a function passed as an argument to another function and executed later. Below are different ways to use callbacks:

---

## **1ï¸âƒ£ Anonymous Function as a Callback**
You can pass an **anonymous function** directly as a callback.

```typescript
function performOperation(num1: number, num2: number, callback: (a: number, b: number) => number) {
    let result = callback(num1, num2);
    console.log("Result:", result);
}

// Passing an anonymous function as a callback
performOperation(10, 5, function (a, b) {
    return a + b;
}); 
// âœ… Output: Result: 15
```

âœ… **Use case**: When the function is simple and used only once.

---

## **2ï¸âƒ£ Arrow Function as a Callback (Shorter Syntax)**
Using an **arrow function** makes the code cleaner.

```typescript

// âœ… Output: Result: 50
```

âœ… **Use case**: When writing short and concise functions.

---

## **3ï¸âƒ£ Named Function as a Callback**

// âœ… Output: Result: 15

âœ… **Use case**: When the function is reusable.

## **4ï¸âƒ£ Inline Named Function as a Callback**

âœ… **Use case**: When you need a specific function only once.

## **6ï¸âƒ£ Callback Inside Another Function (Nested Callbacks)**
You can use multiple callbacks **inside another function**.

```typescript

```

âœ… **Use case**: When calling one function inside another.

## **8ï¸âƒ£ Callbacks with Error Handling**
